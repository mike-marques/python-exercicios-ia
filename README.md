🐍 Exercícios Básicos de Python

Este repositório tem como objetivo oferecer exercícios práticos de Python para iniciantes, com foco em lógica de programação, manipulação de dados e operações básicas. 💻

Os desafios foram pensados para aprender de forma progressiva e permitir que você pratique usando ferramentas como GitHub Copilot ou ChatGPT como apoio. 🤖

🧩 Exercícios Disponíveis

1️⃣ Somando Números

Descrição:
O programa vai receber dois números do usuário e retornar a soma deles.

O que você vai aprender:

Entrada de dados com input()
Conversão de tipos (int ou float)
Operações matemáticas básicas (+)

2️⃣ Calculando o Dobro

Descrição:
O usuário informa um número e o programa retorna o dobro desse valor.

O que você vai aprender:

Operações matemáticas
Armazenamento de valores em variáveis
Manipulação de números inteiros e decimais

3️⃣ Saudações Personalizadas

Descrição:
O programa pede o nome do usuário e imprime uma mensagem de boas-vindas personalizada.

O que você vai aprender:

Manipulação de strings
Concatenar textos
Entrada de dados simples

4️⃣ Verificando Maioridade

Descrição:
O usuário informa sua idade e o programa retorna se ele é maior de idade (18 anos ou mais) ou não.

O que você vai aprender:

Estruturas condicionais (if, else)
Comparação de valores
Lógica booleana

5️⃣ Contando Caracteres

Descrição:
O programa pede uma palavra ou frase e retorna a quantidade de caracteres que ela possui.

O que você vai aprender:

Funções nativas do Python (len())
Manipulação de strings
Trabalhar com entradas de usuário

💡 Objetivo do Projeto:
Criar um ambiente de aprendizado prático, onde iniciantes podem testar soluções, melhorar lógica e explorar Python de forma divertida. Esse projeto também serve como base para contribuições open source: você pode adicionar novos exercícios ou melhorias futuramente. 🚀
